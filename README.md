# Sunrise Mini Project

A simple and beautiful *sunrise animation* webpage created as a *mini project for the Computer Graphics subject*. The project features a calm sunrise scene with animated clouds, a glowing sun, a colorful rainbow, and an inspiring quote.

## Project Description

This project simulates a sunrise with:
- A bright glowing *sun*
- A *rainbow* across the sky
- Soft *clouds* at the bottom
- An inspirational quote: "Every sunrise is a new beginning"
